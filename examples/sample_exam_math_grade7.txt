七年级数学测试卷
考试主题：一元一次方程
满分：100分    时间：60分钟

一、选择题（每题4分，共20分）

1. 下列关于一元一次方程的说法，正确的是（  ）
   A. 方程 x² + 2x - 3 = 0 是一元一次方程
   B. 方程 2x + 1 = 0 是一元一次方程
   C. 方程 x/2 + y = 1 是一元一次方程
   D. 方程 1/x + 1 = 2 是一元一次方程

2. 解方程 3x - 7 = 2，x 的值是（  ）
   A. x = 1     B. x = 2     C. x = 3     D. x = -3/5

3. 若方程 ax + 3 = 0 的解是 x = -1，则 a 的值是（  ）
   A. a = -3    B. a = 3     C. a = 1     D. a = -1

4. 用一元一次方程解决实际问题时，设未知数的原则是（  ）
   A. 只能设问题最终要求的量为未知数
   B. 可以设任何一个未知量为未知数
   C. 必须设两个未知数
   D. 未知数必须是整数

5. 甲比乙大5岁，甲的年龄是乙的年龄的2倍，设乙的年龄为 x 岁，可列方程为（  ）
   A. x + 5 = 2x     B. 2x + 5 = x
   C. x - 5 = 2x     D. x + 5 = x/2

二、填空题（每题5分，共25分）

6. 解方程 2(x + 3) - 4 = 3x - 1，则 x = ________。

7. 若 x = 2 是方程 2x + m = x + 5 的解，则 m = ________。

8. 某班同学去春游，如果每辆车坐45人，则剩余5人没有座位；如果每辆车坐50人，则有一辆车空出10个座位。设共有 x 辆大巴车，列出方程：________。

9. 解方程 (x-1)/2 - (2x+1)/3 = 1，则 x = ________。

10. 一个两位数，十位上的数字比个位上的数字小3，将个位与十位对调后，得到的新数比原数大27，设十位上的数字为 x，则原来的两位数为________。

三、解答题

第11题（10分）
解方程：5x - 3(2x - 1) = 4(x + 2) - 9

第12题（10分）
解方程组（用一元一次方程方法）：
设某数为 x，该数的3倍减去8等于该数加上10，求该数。

第13题（15分）
某工厂计划生产一批零件。如果每天生产120个，则需要比计划多1天完成；如果每天生产150个，则可以提前2天完成。求计划生产的零件总数。

第14题（20分）
小明和小红各有若干本故事书。小明的书的数量是小红的2倍少6本。如果小明给小红12本，则小红的书恰好比小明多2本。
（1）设小红有 x 本书，列出方程。
（2）求小明和小红各有多少本书。
（3）如果再给小红 k 本书，使得两人书的数量相等，求 k 的值。
