课题：一元一次方程的概念与解法
学科：数学
年级：七年级
课时：1课时

一、教学目标
1. 理解一元一次方程的概念，能够识别一元一次方程。
2. 会根据等式性质求解简单的一元一次方程。
3. 能结合具体情境列出简单方程并解释求解过程。
4. 在解题过程中形成检验结果是否合理的意识。

二、教学重难点
重点：一元一次方程的概念、移项与合并同类项的基本步骤。
难点：从实际问题中抽象出方程；移项时符号变化；规范表达求解过程。

三、教学准备
教师准备：PPT、例题讲解步骤卡、课堂练习单。
学生准备：草稿本、练习本。

四、教学过程
1. 情境导入
教师出示问题：小明买了3支笔和1本本子，一共花了11元。已知每支笔2元，本子多少元？
教师提问：如果把本子的价格设为x元，可以怎样表示这个数量关系？
预设学生回答：3乘2加x等于11。
教师板书：3x + 2 = 11，随后追问这个式子对不对，为什么？
教师引导学生发现应为3*2 + x = 11，而不是3x + 2 = 11。

2. 新课讲授
教师给出一元一次方程的定义：只含有一个未知数，未知数的次数是1，等号两边都是整式，这样的方程叫做一元一次方程。
教师展示式子：x + 3 = 5，2x - 1 = 7，x^2 + 1 = 5，x + y = 4。
学生判断哪些是一元一次方程，并说明理由。
教师小结判断标准：一个未知数、次数为1、是等式。

3. 例题讲解
例1：解方程 x + 5 = 12。
教师示范：方程两边同时减去5，得到x = 7。
教师强调：依据是等式的性质，方程两边同时进行相同运算，结果仍然相等。

例2：解方程 3x - 4 = 8。
教师板书：3x = 12，x = 4。
教师口头说明：把-4移到右边就变成+4。
教师提醒学生注意移项后的符号变化。

例3：解方程 2x + 3 = x + 7。
教师提问：先移哪一项更方便？
学生回答后，教师展示步骤：2x - x = 7 - 3，所以x = 4。
教师强调先移含未知数项，再移常数项，也可以根据情况灵活处理。

4. 课堂练习
基础题：
(1) x - 6 = 9
(2) 5x = 20
(3) 4x + 1 = 13

提高题：
(4) 3x + 5 = 2x + 9
(5) 2(x + 1) = 8

应用题：
某数的3倍比这个数大10，求这个数。

教师巡视时重点关注：
- 学生是否能正确移项
- 是否会漏写等号
- 是否会把3x误认为x + 3
- 是否能检验结果

5. 课堂总结
教师总结：
(1) 判断一元一次方程要看未知数个数、次数和是否为等式。
(2) 解方程时要注意等式性质和符号变化。
(3) 解完后最好代入原方程检验。

6. 课后作业
A组（基础巩固）：课本习题中直接求解的一元一次方程3题。
B组（变式提升）：含有未知数在两边的方程2题。
C组（应用拓展）：根据实际情境列方程并求解1题。

五、教学反思预设
1. 部分学生可能会把移项理解成“搬家”，只记结论，不理解依据。
2. 基础薄弱学生可能在合并同类项和符号变化时出错。
3. 中等学生会做例题，但在应用题中不一定能顺利列方程。
4. 能力较强学生完成速度快，需要更有挑战性的变式任务。
